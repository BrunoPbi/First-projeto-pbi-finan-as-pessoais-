# First-projeto-pbi-finan-as-pessoais-
Meu objetivo é aplicar meus conhecimentos adquiridos na faculdade e utilizar ETL, Gestão de projetos, Power BI para visuais , tecnicas de modelagem entre as tabelas e automação de dados para atualizar os visuais em tempo real ou no menor tempo possivel. 

# Planejamento do Projeto – 5W2H
What (O quê?)

Desenvolvimento de um dashboard financeiro pessoal automatizado utilizando Power BI, integrado ao Google Sheets para centralização, análise e visualização de dados financeiros.

Why (Por quê?)

A ausência de uma visão consolidada dos dados financeiros dificulta a tomada de decisão e o controle de gastos. O projeto tem como objetivo melhorar a organização financeira pessoal e aplicar, na prática, conceitos de análise de dados para desenvolvimento profissional e construção de portfólio.

Where (Onde?)

O projeto será desenvolvido em ambiente local, com versionamento e documentação publicados no GitHub.

When (Quando?)

Estruturação dos dados: 19/03/2026
Integração com Power BI: 20/03/2026
Desenvolvimento do dashboard inicial: até 22/03/2026
Versão final do projeto: até 26/03/2026

Who (Quem?)

Projeto desenvolvido individualmente, com atuação em todas as etapas do pipeline de dados: coleta, tratamento (ETL), modelagem, análise e visualização.

How (Como?)

Coleta e armazenamento dos dados via Google Sheets
Conexão automática com Power BI via publicação em formato CSV
Tratamento e transformação dos dados no Power Query
Modelagem dimensional (modelo estrela)
Criação de métricas utilizando DAX
Desenvolvimento de dashboard interativo com indicadores financeiros

How Much (Quanto?)

Custo zero, utilizando ferramentas gratuitas e dados próprios (com versão anonimizada para fins de portfólio).
